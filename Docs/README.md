# Flappy Bird ☁️

## Projektbeskrivning 

Detta är ett webbaserat spel inspirerat av spelet Flappy Bird. I denna version av Flappy Bird styr spelaren en flaxande drake lodrät som ska flyga igenom en serie hinder i spelet. Målet är att undvika dessa hinder samt spelramens kanter. Det gäller att ha en bra tajming för att få draken att flaxa vid rätt tillfälle så att den inte flyger in i något av dess hinder. Denna version går ut på en liknande spellogik samt med en visuellt tilltalande spelupplevelse. Spelet är byggt med JavaScript, HTML och CSS. 

---

## Hur spelar man? 
När spelet börjar börjar påverkas draken direkt av graviation och faller automatisk nedåt. För att hålla draken svävande måste du trycka snabbt på [Pilupp] eller [Mellanslag] för att flaxa genom luften. Uppgiften är att styra draken så att den passerar genom öppningarna mellan hindren utan att krocka med dem. För varje hinder som passeras får man ett poäng. Om draken krockar med hinder eller spelramens kanter tar spelet slut och totala poäng visas. Man kan starta om spelet direkt genom att klicka på uppåtknappen eller mellanslag. 

## Mål ✨
Spelets mål är att klara sig så långt som möjligt genom att passera så många hinder som möjligt. För varje hinder som passerats tilldelas ett poäng. Den slutliga poängen visas under spelets gång och när spelet har avslutats. 

---

## Styrning och funktioner ☁
Spelaren kan använda följande funktioner: 
- Pil upp knappen [↑] eller mellanslag [ ]- får figuren att hoppa/flyga 
- Start-knapp genom knappen [↑] eller mellanslag [ ] för att starta om spelet efter “Game over”
